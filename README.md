# British_Airways_Reviews# ✈️ British Airways Reviews Dashboard

## 📊 Project Overview
This project analyzes customer reviews of British Airways using an interactive Tableau dashboard. The dashboard provides insights into passenger satisfaction across multiple service categories, helping identify trends and areas for improvement.

## 🚀 Dashboard Features

### Key Metrics
- ⭐ Average Overall Rating: 4.2
- 👨‍✈️ Average Cabin Staff Service: 3.3
- 🎬 Average Entertainment: 1.4
- 💺 Average Seat Comfort: 2.9
- 🍽️ Average Food & Beverages: 2.4
- 🏢 Average Ground Service: 3.0
- 💰 Average Value for Money: 2.8

### Interactive Filters
- Review Metric Selection
- Traveller Type
- Date Range
- Seat Type
- Continent
- Aircraft Group

### Visualizations
- Cabin Staff Service Trend by Month
- Cabin Staff Service by Country (Map View)
- Cabin Staff Service by Aircraft Type
- Review Count Analysis

## 🛠️ Tools & Technologies
- Tableau
- Data Visualization
- Data Analytics
- Dashboard Design
- Customer Experience Analysis

## 📈 Insights
- Analyze customer satisfaction trends over time.
- Compare service ratings across countries.
- Identify best-performing aircraft types.
- Understand review distribution and customer feedback patterns.
